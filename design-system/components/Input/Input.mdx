# Input

Le composant `Input` est utilisé pour capturer des données utilisateur.

## Props

- **placeholder** : Texte d'exemple affiché dans le champ.
- **value** : Valeur actuelle du champ.
- **onChange** : Fonction appelée lors de la modification de la valeur.
- **size** : Taille du champ (`small`, `medium`, `large`).
- **backgroundColor** : Couleur d'arrière-plan personnalisée.
